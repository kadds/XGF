基于Direct3D11 的 2D 游戏渲染框架  
==================
**X Game Render Framework**

  
个人学习 Direct3D11 用

1. Scene、Layer、Actor  
2. Batch 渲染器  
3. Action 动画系统  
4. DirectInput （仅支持Mouse、KeyBoard）  
5. UI 系统部分支持： Button、EditText（未完成）、Lable  
6. 透视投影摄像机和正交投影摄像机  
7. 基于Freetype的文本渲染  
  
  

IDE要求 ：VS2015 以上
